# FRDM-KL46Z-PIT-Lab
Assembly code written for the ARM Cortex-M0+ CPU core included on the Freescale Freedom Development Board FRDM-KL46Z.

This code includes:
  Drivers for the UART module on the board.
  Procedures to asynchronously send and recieve data from a serial bus.
  Drivers for the PIT module on the board.
  Procedures to asynchronously run an internal timer with precision of .01s.
  Testing code to time text input provided by a user.
